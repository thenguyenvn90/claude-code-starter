# First Prompts — Build a SaaS Landing Page

Follow these 10 prompts **in order**. Copy each one, paste into Claude Code, and watch it build.

By the end, you'll have a live website deployed on the internet.

---

## Step 1: Set Up the Project

```
Create a Next.js project with TypeScript and Tailwind CSS in this directory.
Use the app router. Set up the basic folder structure.
After setup, run the dev server so I can preview in my browser.
```

## Step 2: Build the Hero Section

```
Create a hero section for the homepage with:
- A large headline: "Ship Faster with AI"
- Subheadline: "The all-in-one platform that helps teams build products 10x faster"
- A primary CTA button: "Start Free Trial" (blue, rounded)
- A secondary link: "Watch Demo →"
- Background: subtle gradient from slate-50 to white
Make it centered and visually appealing.
```

## Step 3: Add Features Section

```
Below the hero, add a features section with 3 cards in a row:
- Card 1: "AI-Powered Code" — "Write, review, and ship code with AI assistance. No more boilerplate."
- Card 2: "Team Collaboration" — "Real-time editing, comments, and version control built in."  
- Card 3: "One-Click Deploy" — "Push to production in seconds. Automatic scaling included."
Each card should have an icon placeholder, title, and description.
Use a light gray background for the section.
```

## Step 4: Add Pricing Section

```
Add a pricing section with 3 tiers side by side:

Free ($0/month):
- 1 project
- Basic AI features
- Community support
- Button: "Get Started"

Pro ($19/month) — mark this as "Most Popular":
- Unlimited projects  
- Advanced AI features
- Priority support
- Team collaboration
- Button: "Start Free Trial"

Enterprise ($49/month):
- Everything in Pro
- Custom integrations
- Dedicated account manager
- SSO & audit logs
- Button: "Contact Sales"

Highlight the Pro plan with a colored border or badge.
```

## Step 5: Add Testimonials

```
Add a testimonials section with 3 cards:

Card 1:
- Quote: "This tool cut our development time in half. We shipped our MVP in 2 weeks instead of 2 months."
- Name: Sarah Chen
- Role: CTO at StartupXYZ

Card 2:  
- Quote: "The AI features are incredible. It's like having a senior developer on the team 24/7."
- Name: Marcus Johnson
- Role: Lead Developer at TechCorp

Card 3:
- Quote: "Best investment we've made this year. ROI was positive within the first month."
- Name: Emily Rodriguez  
- Role: VP of Engineering at ScaleUp

Add a subtle background and good spacing.
```

## Step 6: Add Footer

```
Create a footer with:
- Logo text "ShipFast" on the left with tagline "Build products faster with AI"
- 3 link columns:
  - Product: Features, Pricing, Changelog, API
  - Company: About, Blog, Careers, Contact
  - Legal: Privacy, Terms, Security
- Social media icons row (Twitter, GitHub, LinkedIn)
- Copyright: "© 2026 ShipFast. All rights reserved."
Dark background (slate-900), white text.
```

## Step 7: Make It Mobile Responsive

```
Check the entire page on mobile screen sizes (375px wide).
Fix any layout issues:
- Hero text should stack vertically on mobile
- Feature cards should be 1 column on mobile
- Pricing cards should stack on mobile
- Footer columns should stack on mobile
Add a hamburger menu for mobile navigation if there's a header.
```

## Step 8: Add Smooth Animations

```
Add subtle scroll animations to make the page feel polished:
- Hero section: fade in from bottom on page load
- Feature cards: stagger fade in as you scroll to them
- Pricing cards: slide up on scroll
- Testimonials: fade in on scroll
Use CSS animations or Framer Motion. Keep it subtle, not distracting.
```

## Step 9: Preview and Polish

```
Start the dev server if it's not running.
Look at the full page from top to bottom and fix:
- Any spacing that looks off
- Colors that don't match
- Text that's hard to read
- Images or icons that are missing
- Mobile layout issues
Make it look professional and ready to show to someone.
```

## Step 10: Deploy to Vercel

```
Help me deploy this website to Vercel for free:
1. Make sure the build works: npm run build
2. Walk me through connecting to Vercel
3. Deploy and give me the live URL

If I don't have a Vercel account, tell me how to create one (it's free).
```

---

## What's Next?

After deploying your landing page:

- **Customize the content** — change "ShipFast" to your actual product name
- **Add your own images** — replace placeholder icons with real graphics
- **Connect a domain** — point your custom domain to Vercel
- **Add analytics** — integrate Google Analytics or Plausible
