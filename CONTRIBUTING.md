# Contributing

Thanks for your interest in improving this starter template!

## How to Contribute

1. **Fork** this repo
2. **Create a branch** for your change (`git checkout -b improve-readme`)
3. **Make your changes** and test them
4. **Submit a Pull Request** with a clear description

## What We're Looking For

- Better CLAUDE.md examples or sections
- Improved permission rules in settings.json
- New quality rules in `.claude/rules/`
- Better prompts in FIRST-PROMPTS.md
- Fixes for typos or unclear instructions
- Translations of README or FIRST-PROMPTS

## Guidelines

- Keep it simple. This template is for beginners.
- No framework-specific code. The template should be language-agnostic.
- Test your changes by cloning fresh and running `claude` + `/init`.

## Questions?

Open an [issue](https://github.com/thenguyenvn90/claude-code-starter/issues) or visit [ongboit.com](https://ongboit.com).
