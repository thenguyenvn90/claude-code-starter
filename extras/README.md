# Extras (opt-in)

**EN** — The core (`CLAUDE.md` + `.claude/`) is enough to start. These extras are optional add-ons you copy in
when you want them. Each folder has its own README and links to a full guide on ongboit.com.

**VI** — Phần lõi (`CLAUDE.md` + `.claude/`) đủ để bắt đầu. Các extra dưới đây là tùy chọn, copy vào khi cần.
Mỗi folder có README riêng + link bài hướng dẫn đầy đủ trên ongboit.com.

| Extra | What it does · Làm gì | Guide · Bài |
|---|---|---|
| [`statusline/`](statusline/) | A status bar showing model · %context · git branch. Thanh trạng thái: model, % context, git. | [Status line](https://ongboit.com/claude-code-status-line/) |
| [`hooks/`](hooks/) | A PreToolUse hook that blocks reading `.env`/keys. Hook chặn Claude đọc `.env`/key. | [Hooks](https://ongboit.com/claude-code-hooks/) |
| [`sandbox/`](sandbox/) | A settings preset to run bash sandboxed (fewer Accepts, safer). Preset chạy bash trong sandbox. | [Sandbox](https://ongboit.com/claude-code-sandbox/) |
| [`skills/`](skills/) | One example skill (`commit-helper`). 1 skill mẫu để bạn tự nhân bản. | [Skills](https://ongboit.com/claude-code-skills/) · [Tạo skill](https://ongboit.com/tao-claude-code-skill/) |
| [`subagents/`](subagents/) | One example subagent (`code-reviewer`). 1 subagent mẫu. | [Subagent best practices](https://ongboit.com/claude-code-subagent-best-practices/) |

> Copy only what you need. Each extra's README shows exactly where the files go.
> Chỉ copy cái bạn cần. README từng extra chỉ rõ đặt file vào đâu.
